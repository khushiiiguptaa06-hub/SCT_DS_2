# SCT_DS_2 - Titanic Exploratory Data Analysis

##  Project Overview

This project performs Exploratory Data Analysis (EDA) on the Titanic dataset to identify the factors that influenced passenger survival during the Titanic disaster. The analysis focuses on demographic, socio-economic, and travel-related attributes to uncover meaningful patterns and trends.

---

##  Objective

To analyze passenger data and determine the key factors associated with survival outcomes using statistical analysis and data visualization techniques.

---

##  Tools & Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook / Google Colab

---

##  Dataset

The dataset contains information about Titanic passengers, including:

* Passenger Class (Pclass)
* Name
* Sex
* Age
* Fare
* Embarked Port
* Number of Siblings/Spouses Aboard
* Number of Parents/Children Aboard
* Survival Status

---

##  Analysis Workflow

### Data Preprocessing

* Data inspection and exploration
* Handling missing values
* Data cleaning and preparation

### Exploratory Data Analysis

* Univariate analysis
* Bivariate analysis
* Feature relationship analysis

### Data Visualization

* Survival Distribution
* Survival by Gender
* Survival by Passenger Class
* Age Distribution
* Fare Distribution
* Correlation Heatmap

---

##  Key Findings

* Female passengers had significantly higher survival rates compared to male passengers.
* Passengers traveling in First Class were more likely to survive than those in lower classes.
* Higher ticket fares showed a positive association with survival probability.
* Age and passenger class influenced survival outcomes.
* Socio-economic status played an important role in determining passenger survival.

---

##  Conclusion

The analysis demonstrates that gender, passenger class, and fare were among the most influential factors affecting survival on the Titanic. Through data visualization and exploratory analysis, valuable insights were uncovered regarding the demographic and socio-economic characteristics associated with passenger survival.
